概述

在 VASP 计算中，通过元素替换调控材料性质是常见策略。但有些替换方案 无论怎么优化结构，声子谱总是出现虚频， 意味着该结构在动力学上不稳定。

replace.py 是一个零计算成本的启发式预筛选工具，仅需 POSCAR 文件， 几秒钟即可输出替换元素评分排名。支持双通道运行： 有 pymatgen 时自动使用完整功能，无 pymatgen 时 通过内置 element_data.py 降级运行。 几秒钟即可输出替换元素评分排名。

零成本预筛选

无需 DFT 计算，从化学兼容性原则出发，秒级输出评分排名。

交互模式

直接运行 python replace.py POSCAR，以交互方式选择要替换的原子。

按配位环境分组

同元素不同位点若配位环境不同，自动分组并分别推荐替代元素。

分数氧化态

通过电荷平衡方程自动计算非整数有效氧化态，更准确反映实际电子状态。

加权综合排名

替换同元素全部位点时，按各环境位点数量加权计算综合排名。

候选 POSCAR 输出

直接生成替换后的 POSCAR 文件，配合 --output-dir 一键导出。

自动候选生成

根据元素类型自动筛选候选列表，也支持 `--candidates` 手动指定。

候选 POSCAR 输出

直接生成替换后的 POSCAR 文件，配合 `--output-dir` 一键导出。

JSON 导出

结果可导出为 JSON，便于批量处理和后续自动化流水线对接。

重要提醒：本程序是预筛选工具，不是声子稳定性预测器。 高分意味着化学/几何兼容性较好，声子稳定的概率较高， 但最终仍需 DFT + phonopy 验证。尤其是 Se 替换 S（评分高）仍可能因电子结构原因出现虚频。

交互模式

直接运行 python replace.py POSCAR 即可进入交互模式。 该模式下自动执行以下步骤：

  1. 显示结构摘要 — 文件路径、化学式、所含元素及原子数
  2. 列出所有原子坐标 — 带标签（N1, N2, V1, W1, ...）和 Cartesian 坐标
  3. 按配位环境分组 — 显示每个元素的近邻配位环境、有效氧化态
  4. 等待用户输入 — 支持三种输入格式

输入格式

格式示例含义
元素符号N替换该元素的所有原子，各环境分别推荐 + 加权综合排名
符号+序号N1替换该元素的第 1 个原子（按 POSCAR 顺序）
数字3替换位点 3（从 1 开始编号的传统模式）

自动根据输入决定替换模式，使用体验更流畅。

配位环境分组

同一元素在结构中可能处于不同配位环境。例如，N 原子可能同时处于 VN3 和 VN2W 两种不同的近邻配位中。 replace.py 自动对同元素不同位点进行环境分类：

  • 近邻定义 — 3.5 Å 截断半径内的异种元素（排除同元素近邻）
  • 环境签名 — 按近邻种类和数量生成唯一标识（如 N@V3、N@V2W1）
  • 每个环境独立推荐 — 不同环境可能推荐不同的最佳替代元素

有效氧化态

通过全局电荷平衡方程自动计算目标元素的分数氧化态： 

Sigma(n_i * ox_i) = 0
- 强阴离子固定价态: O=-2, N=-3, F/Cl/Br/I=-1, S/Se/Te=-2, P/As=-3
- 阳离子取常见氧化态
- 目标元素的有效氧化态 = -(总阳离子贡献 + 其他阴离子贡献) / 目标原子数

结果可能是非整数（如 N = -4.33），更真实地反映复杂结构中的电子分布。 可通过 --ox-state 手动覆盖。

原子标签与坐标

交互模式和 --element 模式下，输出结果会按配位环境显示。 每个位点从 1 开始编号，并与 Cartesian 坐标对应，方便可视化软件交叉验证。

评分标准详解

维度权重说明
电子构型 0.45 最重要。同族（相同价电子数 / d 电子数）最高分。 同区块（d↔d / p↔p）次之。电子排布决定成键特性。
离子半径 0.20 Shannon 半径匹配度。基于 exp(-2 × |Delta;r| / r_host)， 差异越小越好。支持分数氧化态插值。
电负性 0.15 Pauling 标度差异。差异小 → 电荷转移少 → 键性相似 → 电子结构变化小。
氧化态 0.10 候选元素是否存在目标氧化态。对分数氧化态自动查找最近整数态并计算兼容度。
周期邻近 0.10 同族 (+0.50) > 同区块 (+0.20) > 邻族 (+0.15) > 同行 (+0.10)。
评分公式：总分 = 电子构型×0.45 + 离子半径×0.20 + 电负性×0.15 + 氧化态×0.10 + 周期邻近×0.10。 权重可通过 --weights 自定义（传入 5 个浮点数， 顺序：半径 电负性 氧化态 周期 电子）。
氧化态自动检测：

检测优先级：

  1. pymatgen — 调用 add_oxidation_state_by_guess()
  2. 分数电荷平衡 — 通过 Sigma;n⊂i;⋅ox⊂i; = 0 计算分数值
  3. 常见价态 — 取 common_oxidation_states[0]

推荐用 --ox-state 手动指定，避免自动检测偏差。

多环境加权综合排名

建议工作流

本程序不能替代 DFT 计算。 评分高 ≠ 声子稳定。电子构型匹配降低了试错成本， 但最终仍需 DFT + phonopy 验证。

完整流程

  1. 启发式预筛选 — 用 replace.py 选出 Top 5–10 候选
  2. DFT 结构弛豫 — 对 Top 5 候选做 VASP 弛豫（ISIF=3）
  3. phonopy 声子计算 — 对弛豫后结构进行 phonon 分析
  4. 结果判读 — 根据声子谱反馈调整：虚频仅出现在 Gamma 点 → 可安全忽略（声学求和规则伪影）；非 Gamma 点虚频 → 真实不稳定

声子不稳定的常见原因

原因表现对策
尺寸失配 高频虚频、结构扭曲 用启发式筛选避免半径差异 >20%
电荷失配 虚频 + 电子不收敛 匹配氧化态；考虑额外掺杂补偿
费米面嵌套 特定 q 点虚频 (π,π) 考虑磁性、自旋极化、U 值
Jahn-Teller 局域结构畸变 检查对称性破缺可能性
声学求和规则 仅 Gamma 点小虚频 可忽略（自动识别为伪影）
后续方向：如果积累更多数据，未来可以引入 ML 势（MACE/CHGNet）替代启发式评分， 进行快速弛豫+声子计算，大幅提高预测准确度。

评分标准详解

维度权重说明
电子构型 0.45 最重要。同族（相同价电子数 / d 电子数）最高分。 同区块（d↔d / p↔p）次之。电子排布决定成键特性。
离子半径 0.20 Shannon 半径匹配度。基于 exp(-2 × |Δr| / r_host)， 差异越小越好。太大 / 太小都会导致晶格畸变。
电负性 0.15 Pauling 标度差异。差异小 → 电荷转移少 → 键性相似 → 电子结构变化小。
氧化态 0.10 候选元素是否存在目标氧化态。完全匹配 → 1.0；±1 → 0.8；±2 → 0.5。 推荐用 --ox-state 显式指定。
周期邻近 0.10 同族 (+0.50) > 同区块 (+0.20) > 邻族 (+0.15) > 同行 (+0.10)。
评分公式：总分 = 电子构型×0.45 + 离子半径×0.20 + 电负性×0.15 + 氧化态×0.10 + 周期邻近×0.10。 权重可通过 --weights 自定义（传入 5 个浮点数）。
氧化态自动检测：支持非整数有效氧化态。 按配位环境分组后，对每组解局域电荷平衡方程 Σ nᵢ·oxᵢ = 0， 阴离子固定价态（O=-2, N=-3, F/Cl/Br/I=-1），过渡金属取常见价态， 目标元素的有效氧化态 = 分数值。也可用 --ox-state 手动指定。
