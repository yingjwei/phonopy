概述

在 VASP 计算中，通过元素替换调控材料性质是常见策略。但有些替换方案 无论怎么优化结构，声子谱总是出现虚频， 意味着该结构在动力学上不稳定。

screen.py 是一个零计算成本的启发式预筛选工具，仅需 POSCAR 文件， 几秒钟即可输出替换元素评分排名。

零成本预筛选

无需 DFT 计算，从化学兼容性原则出发，秒级输出评分排名。

自动候选生成

根据元素类型自动筛选候选列表，也支持 `--candidates` 手动指定。

候选 POSCAR 输出

直接生成替换后的 POSCAR 文件，配合 `--output-dir` 一键导出。

JSON 导出

结果可导出为 JSON，便于批量处理和后续自动化流水线对接。

重要提醒：本程序是预筛选工具，不是声子稳定性预测器。 高分意味着化学/几何兼容性较好，声子稳定的概率较高， 但最终仍需 DFT + phonopy 验证。尤其是 Se 替换 S（评分高）仍可能因电子结构原因出现虚频。

安装

环境要求

组件需求
Python≥ 3.8
pymatgen≥ 2024.0

安装步骤

git clone https://github.com/yingjwei/phonopy.git
cd phonopy
pip install pymatgen

无需其他依赖，直接运行：

python screen.py POSCAR --element V

基本用法

python screen.py <POSCAR> --element <元素符号> [选项]

按元素替换

指定被替换的元素符号，自动找出该元素的所有位点并生成候选列表。

python screen.py POSCAR --element V

手动指定候选

缩小候选范围，只考虑特定元素。

python screen.py POSCAR --element V \
  --candidates Nb,Ta,Mo,W,Cr

指定氧化态

显式指定氧化态，提高评分准确性。

python screen.py POSCAR --element V \
  --ox-state 4

批量输出

生成候选 POSCAR + JSON 结果。

python screen.py POSCAR --element Cl \
  --ox-state -1 \
  --output-dir ./candidates \
  --json results.json

评分标准详解

维度权重说明
电子构型 0.45 最重要。同族（相同价电子数 / d 电子数）最高分。 同区块（d↔d / p↔p）次之。电子排布决定成键特性。
离子半径 0.20 Shannon 半径匹配度。基于 exp(-2 × |Δr| / r_host)， 差异越小越好。太大 / 太小都会导致晶格畸变。
电负性 0.15 Pauling 标度差异。差异小 → 电荷转移少 → 键性相似 → 电子结构变化小。
氧化态 0.10 候选元素是否存在目标氧化态。完全匹配 → 1.0；±1 → 0.8；±2 → 0.5。 推荐用 --ox-state 显式指定。
周期邻近 0.10 同族 (+0.50) > 同区块 (+0.20) > 邻族 (+0.15) > 同行 (+0.10)。
评分公式：总分 = 电子构型×0.45 + 离子半径×0.20 + 电负性×0.15 + 氧化态×0.10 + 周期邻近×0.10。 权重可通过 --weights 自定义（传入 5 个浮点数）。
设计原则：电子构型权重最高（0.45），因为成键特性是决定结构稳定性的首要因素。 相比之下，离子半径和电负性虽重要，但可以通过晶格弛豫部分缓解。

高级选项

自定义权重

# 5 个权重：[电子构型, 离子半径, 电负性, 氧化态, 周期邻近]
python screen.py POSCAR --element V \
    --weights 0.40 0.25 0.15 0.10 0.10

自定义候选列表

逗号分隔的元素列表优先于内置分组。可用任意组合：

python screen.py POSCAR --element S \
    --candidates Se,Te,O

批量生成 + 流水线对接

配合 server_pipeline.py 可完成从筛选到 VASP 弛豫、phonopy 声子计算的全流程自动化：

# 1. 启发式筛选 + 生成候选 POSCAR
python screen.py POSCAR --element V \
    --candidates Nb,Ta,Mo,W,Cr \
    --ox-state 4 --output-dir ./candidates

# 2. 生成 VASP 弛豫作业（server_pipeline）
python server_pipeline.py setup ./candidates \
    -r ./ref_opt -o ./vasp_jobs

# 3. 弛豫完成后生成声子作业
python server_pipeline.py phonon-setup ./vasp_jobs \
    --species "S V Nb"

# 4. 声子计算结果分析
python server_pipeline.py analyze ./vasp_jobs \
    --json phonon_results.json

建议工作流

本程序不能替代 DFT 计算。 评分高 ≠ 声子稳定。电子构型匹配降低了试错成本， 但最终仍需 DFT + phonopy 验证。

完整流程

  1. 启发式预筛选 — 用 screen.py 选出 Top 5–10 候选
  2. DFT 结构弛豫 — 对 Top 5 候选做 VASP 弛豫（ISIF=3）
  3. phonopy 声子计算 — 对弛豫后结构进行 phonon 分析
  4. 结果判读 — 根据声子谱反馈调整：虚频仅出现在 Gamma 点 → 可安全忽略（声学求和规则伪影）；非 Gamma 点虚频 → 真实不稳定

声子不稳定的常见原因

原因表现对策
尺寸失配 高频虚频、结构扭曲 用启发式筛选避免半径差异 >20%
电荷失配 虚频 + 电子不收敛 匹配氧化态；考虑额外掺杂补偿
费米面嵌套 特定 q 点虚频 (π,π) 考虑磁性、自旋极化、U 值
Jahn-Teller 局域结构畸变 检查对称性破缺可能性
声学求和规则 仅 Gamma 点小虚频 可忽略（自动识别为伪影）
后续方向：如果积累更多数据，未来可以引入 ML 势（MACE/CHGNet）替代启发式评分， 进行快速弛豫+声子计算，大幅提高预测准确度。
