1. 概述

替换元素后声子谱不稳定是 2D 材料研究中常见的瓶颈。 本程序从 可迁移的物理化学原则 出发，不依赖特定结构类型， 适用于任意新结构的元素替换预筛选。

设计思路：

  • ❌ 不做"结构匹配"——新结构的声子稳定性无法从旧结构直接推断
  • ✅ 从 POSCAR 的实际键长出发，计算替换后的晶格应变
  • ✅ 基于元素本征属性（半径、电负性、电子构型、氧化态）评分
  • ✅ 数据库仅作为"跨界趋势"参考，不限制于特定结构类型
重要提醒：本程序是预筛选工具，不是声子稳定性预测器。 高分意味着该替换方案在化学和几何上兼容性较好，声子稳定的概率较高， 但最终仍需 DFT+phonopy 验证。

6. 评分标准详解

几何分（权重 0.30）— 最重要

从输入的 POSCAR 直接计算：

  1. 找到被替换原子的最近邻，计算平均键长
  2. 用原子半径估计替换后的预测键长
  3. 计算晶格应变 = |预测键长 − 原键长| / 原键长
  4. 评分公式：exp(-5 × 应变)

应变与声子稳定性的关系：

  • < 3% → 几乎无影响，得分接近 1.0
  • 3–8% → 可能有影响，得分 0.6–0.9
  • 8–15% → 晶格畸变风险高，得分 0.3–0.6
  • > 15% → 很可能不稳定，得分 < 0.3

电子分（权重 0.25）

同族元素 d 电子数相同 → 成键特性最相似：

  • 同族（相同价电子数）：+0.50
  • 同区块（d→d, p→p）：+0.20
  • 还有邻族(+0.15)、同行(+0.10)等加分

半径分（权重 0.15）

基于 Shannon 离子半径。exp(-2 × |Δr| / r_host)，差异越小越好。

电负分（权重 0.10）

Pauling 电负性差异 → 成键特性变化。差异小 → 电荷转移少 → 电子结构变化小。

氧化分（权重 0.10）

与目标氧化态完全匹配 → 1.0；±1 内 → 0.8；±2 内 → 0.5。推荐用 --ox-state 显式指定。

趋势分（权重 0.10）

来自数据库但不匹配具体结构：只看这个元素在所有已测结构中的全局表现：

  • 只出现在稳定列表中 → +0.20（如 V, Cr, Re）
  • 只出现在不稳定列表中 → −0.10（如 Bi, Cu, Sb, W）
  • 既有稳定也有不稳定 → +0.05（稳定性依赖具体组合）
  • 无数据 → 0.00（中性）

8. 重要限制

本程序不能替代 DFT 计算。

声子不稳定的原因非常复杂，本程序只考虑了其中一部分：

本程序考虑的因素本程序未考虑的因素
尺寸失配 → 晶格应变电子关联效应（U 值）
电负性差异 → 键合变化磁性/自旋极化
化学计量/氧化态兼容Jahn-Teller 畸变
价电子数/d电子匹配费米面嵌套
数据库跨界趋势参考电荷密度波/CDW 不稳定性
超胞大小效应/收敛性

建议工作流：

  1. 用本程序预筛 → 选出 Top 5-10 候选
  2. 对 Top 5 候选做 DFT 结构弛豫（ISIF=3）
  3. 对弛豫后结构做 phonopy 声子计算
  4. 根据声子谱结果反馈调整
后续方向： 如果积累更多数据，未来可以用 ML 模型（MACE/CHGNet）替代启发式评分，进行快速弛豫+声子计算， 大幅提高预测准确度（方案 B）。
